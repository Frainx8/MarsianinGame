MarsianinGame

MarsianinGame это программа, написанная на языке C#, для Цифровой олимпиады "Волга-IT'20”. Имеет графический интрерфейс и консольный режим. Читая файл карты, выводит результат на экран и в выходной файл, в котором на первой строке содержит “N” количество ходов, на следующей строке содержится “N” символов перемещений в лабиринте, которые обозначаются как : U(UP), D(DOWN), L(LEFT), R(RIGHT). 

Замечание

Чем больше элементов на карте, тем дольше работает алгоритм.

Установка

Запустить файл setup.exe и следовать инструкции установки. На компьютере должен быть установлен фреймворк .Net core не ниже версии 3.1, который идет в комплекте с установкой.

Использование

Запускать программу от имени администратора. Файл карты может определяется аргументом при запуске из терминала (напр. solution.exe mapFile.txt). Для запуска в консольном режиме, запускать с параметром -console. При отсутствии аргументов, запускается со стандартым именем map.txt. Файлы карт должны находится в папке maps.
